# **Znisa - Portfolio Website**

This project is a complete, fully responsive single-page portfolio website.

---

## âœ¨ **Features**

The website includes the following sections, all built to be responsive across desktop, tablet, and mobile devices:

- **Sticky Navigation Bar:** A header that stays at the top of the page, featuring a responsive hamburger menu for mobile.
- **Hero Section:** A dynamic "above-the-fold" section with a unique grid graphic and a clear call to action.
- **Design Process:** A vertical timeline component detailing the designer's 3-step work process.
- **Featured Projects:** A clean, card-based layout to showcase a portfolio of featured work.
- **Client Testimonials:** A grid of testimonial cards with custom "squircle" border-radius and gradient details.
- **Call to Action (CTA):** A final section encouraging user contact, with an abstract grid graphic.
- **Footer:** A clean footer with the brand logo, social media links, and a "Hire me" button.
